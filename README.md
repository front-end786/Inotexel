# InoTexel.AI Website

Modern, responsive website for InoTexel.AI built with React.

## Features

- 6 Complete Pages:
  - Home (About/Journey page)
  - Contact (with form and map)
  - Projects (Portfolio showcase)
  - Services (Tech solutions)
  - Home Alt (Alternative home design)
  - Careers (Job listings)

- Fully Responsive Design
- Modern UI with Gradient Backgrounds
- Smooth Animations
- React Router for Navigation

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build for Production

```bash
npm run build
```

## Pages Overview

1. **Home** (`/`) - Company journey, stats, and CTA
2. **Contact** (`/contact`) - Contact form with email and location info
3. **Projects** (`/projects`) - Portfolio showcase with project cards
4. **Services** (`/services`) - Tech services and solutions
5. **Home Alt** (`/home-alt`) - Alternative home page design
6. **Careers** (`/careers`) - Job listings and company culture

## Customization

- Colors: Update gradient colors in CSS files
- Content: Edit text in respective page components
- Images: Replace placeholder backgrounds with actual images
- Logo: Update logo in Navbar component

## Technologies Used

- React 18
- React Router DOM 6
- CSS3 with Flexbox & Grid
- Modern ES6+ JavaScript
